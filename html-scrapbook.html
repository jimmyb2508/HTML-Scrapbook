HTML Scrapbook

Block Elements

  • Start on a new line
  • End with a new line
  • Take up a much width as they can

Document Metadata

<head> <meta> <style></style> <title>

Forms

<form> <fieldset>

Headings

<header> <hgroup> <h1>I am an h1</h1> <h2>I am an h2</h2> <h3>I am an h3</h3> <h4>I am an h4</h4> <h5>I am an h5</h5> <h6>I am an h6</h6> <footer> <menu> <menuitem>

Main Root

<html>

Links

<nav> <link>

Text Content

<address> <article> <aside> <blockquote> <body> <details> <dialog> <div> <dir> <figure> <figcaption> <hr> <main> <p> <pre> <section> <summary>

Lists

  • Unordered Lists:
      • <ul> <li></li> </ul>
  • Ordered Lists:
      • <ol> <li></li> </ol>
  • Description Lists:
      • <d1> <dt> <dd> </dd> </dt> </d1>

Tables

<caption> <col> <colgroup> <table> <tbody> <td> <thead> <th> <tfoot> <tr>

Inline Elements

  • Stay on the same line as their siblings
  • Take up only as much room as they need

Buttons

<button>

Demarcating Edits

<del> <ins>

Embedded Content

<applet> <embed> <iframe> <noembed> <object> <param>

Forms

<datalist> <input> <label> <legend> <select> <meter> <optgroup> <option> <output> <progress> <textarea>

Inline Text Semantics

<abbr> <acronym> <b> <bdi> <bdo> <big> <br> <cite> <code> <data> <dfn> <em> <i> <kbd> <mark> <q> <quote> <rb> <rtc> <ruby> <s> <samp> <slot> <small> <span> <strong> <sub> <svg></svg> <sup> <template> <time> <tt> <u> <var> <wbr>

Image and Multimedia

<area> <audio> <img> <map> <picture> <source> <track> <video>

Links (or anchors)

<a>

Scripting

<canvas> <noscript> <script>
